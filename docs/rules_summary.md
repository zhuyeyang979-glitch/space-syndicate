# 《太空辛迪加》规则速读

完整桌游规则书见：[docs/tabletop_rulebook.md](tabletop_rulebook.md)。

## 你怎么赢？

推进城市化、经营商品和商路、保护自己的收入份额，并破坏竞争者的收入。当任意玩家达到本局现金目标时，终局倒计时开始；倒计时结束后现金最多者获胜。现金归零会提前失败。

## 开局先做什么？

每局 3-8 席，通常是 1 名真人玩家对 2-7 名 AI。所有玩家公开选择不重复的角色，并选择一张 I 级起始怪兽牌。游戏开始后，先把起始怪兽召唤到星球上；有怪兽后，你才能从怪兽所在区或相邻区购买区域牌，并寻找城市化牌来建立收入份额。

## 钱从哪里来？

城市化份额是主要收入来源。推进城市化不是基础动作，而是通过城市化牌完成；每次推进会新增一种商品生产、新增一种商品需求，或提升该区域通商速度。城市 GDP 来自生产、需求、运输速度、商路、合约和商品价格；GDP 会按秒转化为现金，并按对应城市化份额分配。

## 城市化份额怎么看？

一个区域的城市化可以由多个玩家同时或轮流推进。鼠标悬停在区域上时，会用饼状图显示你在该区域城市化中的份额，以及其他玩家合计份额；你不会直接知道对手分别是谁、分别占多少、或有几个人参与。

## 怎么买牌？

双击区域可以查看该区域牌架。怪兽在该区域或相邻区域时，才可以从该区域买牌；怪兽所在区通常打折，相邻区按原价。普通手牌上限为 5 张，超出时需要私下弃掉一张已有牌。重复获得同名牌会升级，等级为 I-IV。

## 怎么出牌？

打牌时按卡牌要求选择目标，例如区域、怪兽、玩家、城市、商品或两个合约区域。多数卡牌需要满足商品流动条件；满足条件不等于消耗商品。卡牌提交后会公开展示并进入牌轨，但出牌者默认隐藏。

## 公开牌轨和竞价有什么用？

所有打出的牌都会留在公开牌轨上。多名玩家几乎同时出牌时，会进入竞价，公开金额决定结算顺序。牌轨、竞价金额、目标和结算结果都是推理线索。你可以猜某张牌是谁打出的；猜对赢钱并公开贴标签，猜错付钱。

## 怪兽为什么重要？

怪兽会自动移动、破坏、吸取资源和攻击其他怪兽。玩家不能持续指挥怪兽，只能用卡牌、诱导和绑定技能影响它们的一次行动。怪兽受伤时，归属玩家会损失现金，这会形成公开线索。

## 商品和商路为什么重要？

商品价格来自供需、路线、合约、天气、怪兽压力和仓储压力。商路连接生产地和需求地，路线受损会影响相关城市收入。城市化份额和具体商品绑定：你为某区域新增某商品生产、需求或通商速度，就获得该项目的贡献份额。合约也按商品判断，由该商品在目标城市中的控制者决定是否签约。

## 特殊系统速览

| 系统 | 你需要知道什么 |
| --- | --- |
| 角色 | 开局公开选择，提供不同经济、情报、怪兽、军队或购牌优势 |
| 城市化 | 通过卡牌新增生产、需求或通商速度，并获得对应收益份额 |
| 合约 | 让两个区域或城市建立交易关系，由相关商品控制者决定是否签约 |
| 情报 | 用公开线索和情报牌推测城市、卡牌、怪兽归属 |
| 军队 | 更可控但更短暂的战斗力量，可移动、防守、打击或攻击怪兽 |
| 怪兽赌局 | 怪兽战斗前全员公开下注，按现金百分比下注，赢家分赌池 |
| 天气 | 提前预报，会影响生产、运输、消费、怪兽和金融风险 |
| 金融 | 可买涨/做空城市 GDP 或商品价格，按真实变化和相关城市化份额结算 |

## 第一局建议

先召怪兽，再寻找或购买第一张城市化牌，为一个有商品潜力的区域新增生产、需求或通商速度。之后观察公开牌轨、区域饼图和 GDP 变化：谁在赚钱、谁在破坏、谁可能占有哪条商品路线，才是这局游戏的核心。
