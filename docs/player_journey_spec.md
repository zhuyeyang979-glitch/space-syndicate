# Hearthstone-grade Player Journey v1

目标：玩家第一次打开《太空辛迪加》时，不需要读长规则，也能沿着“主菜单 → 新手战役 → 教学目标 → 实战 → 奖励 → 复盘 → 下一关”的路径持续玩下去。

## 前 30 分钟路径

| 时间 | 玩家看到 | 目标 |
| --- | --- | --- |
| 00:00 | 星球赌桌主菜单 | 三个主入口：新手战役、快速开局、资料库 |
| 00:30 | 新手战役入口 | 看到当前进度、下一关、已解锁关卡 |
| 01:00 | 关卡 Briefing | 只读目标、预计时间、奖励、开始按钮 |
| 02:00 | 主桌 Campaign/Scenario Coach | 一步一目标，不遮挡星球、手牌、牌轨、竞价板 |
| 03:00 | 点区 | 选择推荐区域 |
| 04:00 | 首召 | 打出起始怪兽，开放附近牌架 |
| 05:00 | 建城 | 建第一座城市，建立现金流 |
| 06:00 | 牌架 | 打开区域牌架，比较可买牌 |
| 07:00 | 买牌 | 完成一次购买或换购 |
| 08:00 | 出牌 | 提交匿名牌，看到公开牌轨 |
| 09:00 | 公开牌轨 | 查看右侧详情和公开线索 |
| 10:00 | 情报档案 | 练习一次归属推理 |
| 12:00 | 关卡成功 | 弹出奖励和解锁 |
| 13:00 | 复盘 | 看到关键行动、学到什么、下次建议 |
| 14:00 | 继续动机 | 明确新徽章、解锁内容和下一关主题 |
| 15:00 | 下一关 | 推荐继续到下一关 |
| 20-30 分钟 | 毕业挑战 | 进入 4 人 PVE 小局 |

## 产品原则

- 主菜单不讲开发历史，只给玩家下一步。
- 关卡页不讲长规则，只讲“这一关要做什么”。
- 主桌 Coach 永远只有一个主 CTA。
- 奖励不涉及真钱、抽卡包或付费，只做本地解锁、徽章、复盘评分。
- 任何玩家可见 UI 不显示对手真实现金、真实手牌、AI 私有计划、匿名牌真实归属。

## 完成定义

本版本完成后，测试者可以从主菜单进入“新手战役”，选择 10 个关卡中的已解锁关卡，查看 briefing，进入对应剧本，完成目标后看到奖励/解锁和复盘，并继续下一关。
