# Scenario Reward Spec

奖励系统只服务“继续玩”的动机，不做付费、不做抽卡、不做复杂养成。

## 奖励类型

- `unlock_chapter`：解锁下一关。
- `unlock_codex`：解锁图鉴条目或推荐阅读。
- `unlock_role`：解锁推荐角色说明。
- `unlock_scenario`：解锁练习剧本。
- `badge`：本地徽章。
- `recap_score`：复盘评分。

## RewardPanel 显示

- 关卡完成标题。
- 用时。
- 完成目标数。
- 错误次数。
- 查看提示次数。
- 新解锁。
- 下一关按钮。
- 复盘按钮。

## 隐私边界

奖励和复盘只可使用公开日志与当前玩家自己的私密日志，不显示对手真实现金、手牌、AI 私有计划或匿名牌真实归属。
