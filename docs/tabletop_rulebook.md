# 《太空辛迪加》试玩规则书 v0.4

这是一份面向真人测试者的桌游规则书。你不需要读代码，只要按这份说明理解一局游戏如何开始、如何赚钱、如何用牌、如何读线索，以及如何获胜。

## 1. 游戏概念

《太空辛迪加》是一款实时进行的隐藏信息商业怪兽牌局。玩家扮演外星辛迪加，在同一颗星球上暗中推进城市化、经营商品、操纵商路、打出公开展示但归属隐藏的卡牌，并利用怪兽、军队、合约和情报系统争夺最终财富。

这不是普通的“抽牌打怪”游戏。你真正要做的是：

- 推进城市化，获得城市 GDP 份额。
- 保护自己的城市和商路。
- 破坏竞争者的城市、商品路线和金融仓位。
- 通过公开线索推理谁做了什么。
- 在终局倒计时结束时拥有最多现金。

## 2. 游戏目标

一局游戏的胜利条件很简单：

当任意玩家达到本局现金目标时，游戏进入终局倒计时。倒计时结束后，现金最多的玩家获胜。

如果玩家现金归零，该玩家会提前失败并退出本局。精确平局在试玩版中按并列处理。

## 3. 玩家人数与信息边界

每局有 3-8 个席位，通常是 1 名真人玩家对 2-7 名 AI 对手。

### 公开信息

所有玩家都能看到：

- 星球地图、区域、城市化外观和破坏状态。
- 商品价格、供需趋势、商路和天气预报。
- 公开牌轨上的卡牌、出牌结果、竞价金额和赌局下注。
- 已经被揭示的城市化份额、卡牌或怪兽归属。
- 角色牌，因为角色是开局公开选择的身份。

### 私有信息

只有你自己能看到：

- 你的准确现金。
- 你的手牌和弃牌选择。
- 你在各区域城市化中的真实份额。
- 你的私有标注和推理。

你看不到对手的准确现金、手牌、弃牌和私有计划。你也不会直接知道某个区域里有几个对手持有城市化份额。你只能通过公开行动、商品条件、竞价、怪兽损伤、城市收益变化和情报牌推测他们。

## 4. 游戏组件

### 星球地图

游戏发生在一颗随机生成的球形星球上。近距离时，你看到的是局部平面投影；拉远时，你会看到完整星球。

星球被划分为多个区域。区域可能是陆地、海洋或特殊地貌。陆地更适合城市化，海洋常承担运输、海洋商品和航路压力。

### 区域

每个区域通常包含：

- 地形。
- 区域生命值。
- 已建立的生产商品。
- 已建立的需求商品。
- 通商速度和商路状态。
- 可提供的区域牌架。
- 商路、天气、怪兽吸引度和破坏记录。

区域被严重破坏后，会影响城市收入、商路和商品流通。

### 城市化

城市化是主要收入来源。一个区域可以有一条共同的城市化进程，不是某个玩家独占的一座城市。多个玩家可以同时或轮流推进同一区域的城市化。

推进城市化不是免费基础动作，而是通过城市化类卡牌完成。每次成功推进城市化，通常会在该区域新增或强化一种内容：

- 新增一种商品生产。
- 新增一种商品需求。
- 提升该区域的通商速度。

城市化没有固定上限。区域越被推进，越可能拥有更多商品生产、更多商品需求和更强运输能力。城市化会在地图上公开呈现，但具体由谁推进、各自占多少份额默认隐藏。

### 城市化份额

每个玩家在一个区域的城市化中可能拥有不同份额。鼠标悬停在区域上时，界面会用饼状图显示：

- 你在该区域城市化中的份额。
- 其他玩家合计份额。
- 已公开揭示的特殊归属标签。

你不会直接看到对手分别是谁、分别占多少，也不会知道一共有几个对手参与了该区域城市化。

商品流通带来的收益，会按照玩家在该区域相关城市化份额中的比例分配。买涨、做空、合约奖励、仓储协议或其他以该区域城市化为标的的收益，也会按对应份额结算，除非卡牌写明有特殊分配方式。

### 商品绑定

城市化份额不是一个抽象总股份，而是和具体商品、需求或通商项目绑定。

例如：

- 你打出一张城市化牌，为某区域新增“星鳍鱼群生产”，你就获得该区域星鳍鱼群生产项目的贡献份额。
- 你打出一张城市化牌，为某区域新增“潮汐电浆需求”，你就获得该需求项目的贡献份额。
- 你打出一张城市化牌，提升该区域通商速度，你就获得该通商项目的贡献份额。

一个商品在该城市 GDP 中贡献多少，就决定了相关贡献者能分到多少该商品带来的收入。合约也按商品判断：如果一张合约要连接某个商品的供给或需求，由该商品在对应城市中的控制者决定是否签约。

城市通过生产、消费、运输和合约获得 GDP。GDP 会持续转化为现金收入，并按城市化份额分配给对应玩家。

### 商品

商品是经济系统的核心。一个区域可能生产某种商品，也可能需求某种商品。商品价格会根据供给、需求、商路、合约、天气、怪兽压力和仓储压力变化。

海洋也可以生产商品，例如星鳍鱼群、巨藻纤维、海底黑油、潮汐电浆等。只有本局星球实际存在的商品，才会带入相关商品卡和区域牌架。

### 卡牌

卡牌是玩家干预局势的主要方式。卡牌包括怪兽、城市经营、商品、金融、合约、情报、军队、天气、直接互动等类型。

卡牌等级使用罗马数字 I-IV。重复获得同名卡时，会自动升级到更高等级，最高为 IV。

### 角色牌

角色牌代表玩家扮演的外星势力。角色在开局公开选择，每局不重复。角色会改变你的开局优势、经济路线、情报方式、购牌范围或特殊能力。

角色不会绑定你的起始怪兽，因此别人不能只通过角色判断哪只怪兽属于你。

### 怪兽

怪兽由怪兽牌召唤。怪兽有生命值、移动速度、行动概率、偏好商品、偏好地形和在场时间。

怪兽不是玩家持续控制的单位。它们会根据自己的生态和行动表自动移动、破坏、吸取资源或攻击其他怪兽。玩家可以通过卡牌、诱导和绑定技能影响它们的一次行动。

### 军队

军队由军队牌部署。军队比怪兽更弱、更短暂，但更可控。常见军队包括战斗机、轰炸机、坦克、导弹阵地、潜航舰队和星海战舰。

军队通过命令牌执行移动、防守、攻击怪兽或打击区域等简单指令。

### 公开牌轨

所有打出的卡牌都会进入屏幕上方的公开牌轨。牌轨记录已经打过、正在展示、等待结算的卡牌。

你可以查看牌轨，推测某张牌是谁打出的，并对其归属下注或做标记。猜对会赢钱，并让该牌贴上公开归属标签；猜错会付钱给真正出牌者。

## 5. 开局设置

1. 选择总席位数：3-8 席。
2. 选择 AI 数量：2-7 个 AI 对手。
3. 选择本局挑战深度。挑战深度会影响星球大小、区域数量和现金目标。
4. 每名玩家公开选择一个不重复的角色。AI 可随机分配角色。
5. 每名玩家选择一张 I 级起始怪兽牌。
6. 所有玩家获得基础初始资金；角色可能公开修改自己的初始资金。
7. 游戏开始后，玩家首先要召唤起始怪兽。

起始怪兽的第一次召唤没有区域限制，也不需要 GDP 份额条件。之后打出高等级怪兽牌时，通常要满足牌面写明的区域和 GDP 份额条件。

## 6. 一局游戏的基本流程

游戏是实时进行的，不按回合收入结算。你可以把一局理解为不断循环的六个动作：

1. 首召怪兽。
2. 获取并打出城市化牌。
3. 从区域牌架购买卡牌。
4. 打出卡牌影响地图、经济或对手。
5. 观察公开牌轨、竞价和结算结果。
6. 根据 GDP、商品、怪兽和情报线索调整路线。

你不需要每秒都操作。更像桌游的是：观察桌面，选择一个明确动作，然后等待结果产生新的线索。

## 7. 城市、GDP 与赚钱

城市化份额是你最稳定的现金来源。

城市 GDP 主要来自：

- 生产：城市或区域能产出多少商品。
- 需求：城市能消耗或吸引多少商品。
- 运输：商品通过区域和商路流动的速度。
- 商路：生产地和需求地之间是否连通。
- 合约：城市之间是否建立了有利交易。
- 竞争：经营相似商品的城市会抢客户。
- 损伤：城市、区域或商路被破坏会降低收入。

GDP 会按秒转化为现金。系统先计算该区域由各商品、需求和通商项目产生的 GDP，再按照对应城市化份额分给玩家。怪兽破坏、军队打击、路线断裂或商品价格下跌，都会让相关城市的 GDP 下降。金融和做空类卡牌也会利用这些变化。

你的目标不是“抢到一座完整城市”，而是在多个区域、多个商品项目中建立有价值的城市化份额，并在关键时刻保护它们。

## 8. 商品、供需与商路

每个区域通常从少量生产和需求开始。后续玩家可以通过城市化牌或合约牌增加、替换或删除生产与需求，也可以提高该区域通商速度。

商品价格由供需和压力决定：

- 供给过多会压低价格。
- 需求、合约和稀缺路线会抬高价格。
- 商路受损会降低流通效率。
- 天气会影响生产、运输或消费。
- 怪兽偏好的商品会更容易成为怪兽目标。
- 仓储和期货会制造额外金融风险。

商路是商品从生产地到需求地的运输路径。玩家可以查看某种商品的商路，从而判断哪些区域是经济命脉。因为城市化份额和商品绑定，所以同一座城市中，不同商品路线可能对应不同玩家的收益和签约权。

全局市场会定期公开刷新供需和价格趋势。现金收入仍然按秒进行，市场刷新只是公开重算和展示大局变化。

## 9. 区域牌架、购牌与手牌

每个区域都有自己的区域牌架。双击区域即可查看该区域提供哪些卡牌，即使当前不能购买也可以查看。

每个区域的牌架由两种固定牌位和若干变化牌位组成：

- 一张绑定本区商品的城市发展牌。陆地区域会按生产、需求或通商重点提供项目；海洋区域优先提供适合海运经济的本地商品项目。
- 一张与本区生态和商品偏好尽量匹配的怪兽牌。不同区域优先提供不同怪兽。
- 其余牌位从本局星球实际存在的商品及可用卡池中生成。

固定牌位让区域具有长期战略身份：你可以提前规划要经营哪种商品、争夺哪只怪兽，而不是等待整副牌架完全随机刷新。

### 什么时候可以买牌？

默认情况下，只有当怪兽位于该区域或相邻区域时，玩家才能从该区域购买卡牌。

- 怪兽所在区域：通常有折扣。
- 怪兽相邻区域：通常按原价购买。
- 角色或卡牌可能扩展购牌范围，但会有额外条件或价格倍率。

购买资格在打开购买窗口时检查。只要你打开窗口时满足条件，即使怪兽随后离开，也可以继续完成这次购买。

同一名玩家同时只能打开一个区域牌架购买窗口。

### 手牌上限

普通手牌上限为 5 张。购买新牌会花钱。

如果买入会超过手牌上限，你必须先私下选择一张已有牌弃掉，才能完成购买。你的手牌数量、弃牌选择和弃掉的牌都是私有信息。

怪兽或军队提供的固定技能牌不占普通手牌上限。

### 重复卡牌与升级

重复获得同名卡时，会自动合成为更高等级：

- I + 同名牌 → II
- II + 同名牌 → III
- III + 同名牌 → IV

卡牌最高 IV 级。购买价格通常以 I 级价格为基准，升级带来更强效果、持续时间或更完整路线。

## 10. 出牌、公开牌轨与竞价

打出卡牌时，系统会根据卡牌要求询问目标。例如目标可能是一个区域、一只怪兽、一名玩家、一个城市、一种商品或两个合约区域。

大多数 I 级牌不需要经济门槛，城市发展牌也始终可以直接打出。更高等级或高影响卡牌通常要求你在一个区域拥有足够 GDP 份额：

- 普通路线通常为 I 级 0%、II 级 15%、III 级 25%、IV 级 35%。
- 强力玩家互动、夺取或全局效果通常从 I 级 10% 起步，最高可到 IV 级 40%。
- 牌面会写明检查目标区域、合约发起区，还是你的任一经营区。

GDP 份额是出牌资格，不会因为打牌而减少。商品名称用于决定一张牌会出现在哪些区域牌架，并帮助其他玩家推理来源；它不再是所有卡牌通用的出牌消耗。有些卡牌仍会另外支付牌面写明的现金费用。

### 提交即承诺

一次性卡牌进入公开队列后会离开手牌。即使后续被反应牌阻止，或结算时条件失效，通常也不会返回手牌。

### 公开展示

卡牌进入队列后，会在全场展示一小段时间。所有人都能看到这张牌、目标和公开条件，但看不到出牌者。

### 同时出牌与竞价

如果多个玩家在很短时间内提交卡牌，这些牌会进入同一批次竞价。竞价金额公开，但出价者和出牌者的身份默认不公开。

竞价决定该批卡牌的结算顺序。给出的小费会支付给上一张正在结算卡牌的真正出牌者，因此竞价本身也可能成为推理线索。

没有明显最高价时，按座位顺序处理。

## 11. 卡牌类别

试玩版主要卡牌类别如下：

| 类别 | 做什么 |
| --- | --- |
| 怪兽牌 | 召唤、升级或刷新怪兽，提供绑定技能 |
| 绑定技能 | 由怪兽或军队提供，可重复使用，有冷却 |
| 城市发展 | 绑定区域本地商品，新增生产、需求或提升通商速度 |
| 城市经营 | 放大已有城市化项目的收入、效率或防御 |
| 城市压制 | 降低生产、需求、交通或破坏商路 |
| 商品操作 | 改变供需压力、商品路线或价格趋势 |
| 金融/期货 | 买涨、做空、囤货、套保或押注 GDP 变化 |
| 合约 | 让两个区域或城市建立交易关系 |
| 情报 | 查看、追溯或缩小归属嫌疑范围 |
| 军队 | 部署可指挥战斗单位或下达命令 |
| 天气 | 查看、改写或利用天气窗口 |
| 直接互动 | 干扰玩家手牌、城市归属或关键行动 |

卡牌详情页会显示费用、等级、目标、关键词和升级梯度。主桌上只显示最关键的信息；完整解释在图鉴和规则附录中查看。

## 12. 怪兽规则

怪兽是地图压力的主要来源。

每只怪兽都有：

- 生命值。
- 移动速度。
- 在场时间。
- 地形适应性，例如陆行、海栖、飞行、轨道或混合。
- 偏好商品或地形。
- 自动行动表。
- 攻击伤害、击退距离和区域破坏能力。

怪兽会根据地形、商品、城市、商路、其他怪兽、天气和诱导效果自动选择目标。它们不是简单冲向最近城市。

### 怪兽归属

怪兽由某名玩家召唤，但归属初始隐藏。怪兽受伤时，召唤者会按损伤比例失去现金，这会形成公开线索。某些情报或推理结果可以进一步揭示怪兽归属。

### 怪兽升级

如果场上已有你拥有的同名怪兽，再打出同名怪兽牌通常会升级或刷新它，而不是召唤第二只。升级会提高生命值、刷新在场时间，并让自动行动表更倾向高阶招式。

### 绑定技能

召唤或升级怪兽会给拥有者固定技能牌。技能牌可以重复使用，但有冷却。它们只能影响一次具体行动，例如诱导移动、强化攻击、刷新状态或短暂改变目标。

### 移动与击退

怪兽移动、攻击和击退都按星球表面的距离演出，不是瞬间跨区域。飞行怪兽通常移动更快，且飞行时不会造成普通践踏破坏；海栖怪兽在海洋中更快，离开适合地形会变慢。

## 13. 怪兽赌局

当多只怪兽相遇并即将战斗时，可能触发怪兽赌局。

赌局发生时，整局游戏时间冻结，所有玩家在最多 30 秒内公开下注。所有玩家必须选择一个参战怪兽或参战阵营下注，不能观望不下注。

### 下注方式

赌局基础下注不是固定金额，而是每名玩家当前现金的一定比例。基础比例会在 5%-10% 之间波动。之后玩家可以按 1% 为单位加注。

下注金额和玩家身份公开，因此赌局既是翻盘机会，也是重要线索：别人愿意押谁、押多少，可能暗示他关心哪只怪兽或哪个区域。

### 结算

战斗结束后，以实际造成最多伤害的怪兽或阵营为赢家。押中者平分赌池。多人混战时，可以有多个可下注目标。

这个系统让现金较少的玩家仍有机会通过判断战斗局势追回差距。

## 14. 军队规则

军队是玩家可以更直接指挥的短期战斗力量。

每名玩家通常最多同时拥有一支军队。角色能力可能提高上限。

军队不会像怪兽那样自动乱跑。它们通过命令牌执行简单动作：

- 前往指定区域。
- 防守指定区域。
- 打击指定区域。
- 攻击指定怪兽。
- 控制或保护关键商路。

军队移动本身不会造成怪兽式践踏破坏，但攻击、封锁和军事压力会影响区域生命值、商路或城市 GDP。不同军队适应不同地形：战斗机适合空中机动，坦克适合陆地，潜艇和军舰适合海洋。

## 15. 合约规则

合约牌让两个区域或城市建立交易关系。合约通常涉及一种或多种商品。

打出合约牌时，玩家先选择供应区域和需求区域。卡牌公开展示后，系统会根据合约涉及的商品，找到该商品在目标城市中的控制者，并把签约选择发给这个控制者。签约选择只有目标控制者能操作，但合约牌本身和公开条件所有人都看得到。

合约可能带来：

- 新增生产或需求。
- 替换商品类型。
- 增加商路速度。
- 签约奖励。
- 拒签惩罚。
- 商品升级或需求扩张。

合约的双方身份默认不公开。其他玩家可以从商品条件、路线变化、城市收益和情报牌中推测。由于签约权按商品控制者决定，同一区域里的不同商品合约，可能由不同隐藏玩家负责选择是否签约。

## 16. 情报、标注与归属竞猜

推理是本游戏的重要部分。

你可以给区域或城市做私有标注，例如“我怀疑这是 2 号玩家的城市”。这些标注只对你可见。

你也可以对公开牌轨上的卡牌进行归属竞猜：

- 猜对：真正出牌者付钱给你，该牌贴上公开归属标签。
- 猜错：你付钱给真正出牌者，但归属不会因此公开。

情报牌可以帮助你查看城市归属、追溯历史出牌、追踪合约关系或缩小嫌疑范围。情报通常提供线索，而不是让整局隐藏信息完全消失。

## 17. 天气与新闻

天气是公开预报系统。星球会提前显示未来天气窗口，通常提前约 1-3 分钟。一次天气会影响少量区域，具体数量随星球大小变化。

天气可能影响：

- 商品生产。
- 商品运输。
- 城市消费。
- 怪兽移动或目标倾向。
- 金融和商品价格风险。

玩家可以通过天气类卡牌改变或利用天气。

新闻和事件不作为无缘由的被动随机惩罚出现。新闻类效果来自玩家打出的卡牌，例如制造市场传闻、危机报道、热度操纵或监管风暴。它们和其他卡牌一样公开展示，但归属默认隐藏。

## 18. 金融、期货与 GDP 买涨买跌

金融牌让玩家把经济变化转化为现金。

常见金融路线包括：

- 买某个区域或商品项目的 GDP 上涨。
- 做空某个区域或商品项目的 GDP 下跌。
- 买某种商品价格上涨或下跌。
- 囤积商品等待价格变化。
- 给城市或商路购买保险。
- 对冲怪兽或路线破坏风险。

这些牌通常有明确时间窗口，例如一段时间内比较价格或 GDP 的变化。收益来自真实经济变化，而不是直接任意改价。若金融协议绑定某个区域城市化或某个商品项目，收益和损失按相关城市化贡献份额分配。

囤货通常需要仓储城市。仓储会提高该城市的战略价值，也让它更容易成为怪兽、军队或做空牌的目标。

## 19. 游戏结束

当任意玩家达到本局现金目标时，终局倒计时开始。倒计时期间所有玩家仍可继续打牌、破坏、保护、下注和冲刺。

倒计时结束后，按现金排名，现金最多者获胜。

玩家现金归零会提前失败。若所有区域被毁，游戏会立即结算。

## 20. 第一局建议

第一次试玩时，建议按这个顺序阅读桌面：

1. 看中央星球：哪里有城市、怪兽、商路和天气。
2. 看自己的现金和目标进度：你是在扩张、追赶还是保命。
3. 看手牌：先找“城市化”“赚钱”或“买牌”的牌。
4. 双击区域：查看区域牌架，理解这片区域提供什么路线。
5. 看公开牌轨：最近谁在打什么类型的牌。
6. 看 GDP 和商品趋势：找出哪里正在赚钱，哪里正在被破坏。

### 第一局推荐动作

1. 先召唤起始怪兽。
2. 选择一个有商品潜力的区域。
3. 打出或购买第一张城市化牌，新增生产、需求或通商速度。
4. 打开怪兽所在区或相邻区的区域牌架，继续补强这条路线。
5. 观察公开牌轨，试着猜别人为什么打那张牌。
6. 当怪兽靠近重要城市时，决定是保护、诱导、下注，还是做空。

### 常见误区

- 只盯着手牌，不看城市化份额和城市 GDP。
- 只想打别人，不保护自己的商路。
- 购买很多当前无法满足商品流条件的牌。
- 忽视竞价金额和怪兽受伤带来的归属线索。
- 在没看商品供需前就盲目推进城市化。
- 把怪兽当作稳定部队使用；怪兽会自动行动，玩家只能影响它们。

## 21. 关键词速查

| 关键词 | 含义 |
| --- | --- |
| GDP | 城市当前经济产出，会持续转化为现金 |
| 城市化牌 | 推进区域城市化的卡牌，可新增生产、需求或通商速度 |
| 城市化份额 | 玩家在区域某项城市化贡献中的收益比例 |
| 商品流动量 | 商品从生产到需求的规模 |
| 商品流动速度 | 商品通过区域和商路的速度 |
| 区域牌架 | 每个区域提供的可购买卡牌 |
| 公开牌轨 | 记录所有公开打出的卡牌和结算顺序 |
| 竞价 | 多张牌同时提交时，用公开金额决定结算顺序 |
| 小费 | 竞价金额，可能支付给上一张结算牌的出牌者 |
| 归属竞猜 | 猜某张牌或城市属于谁的推理下注 |
| 绑定技能 | 怪兽或军队给拥有者的固定可重复技能 |
| 怪兽赌局 | 怪兽战斗前的公开百分比下注 |
| 商路 | 商品从生产地到需求地的运输路径 |
| 商品控制者 | 某商品在对应城市化项目中的当前控制玩家，负责相关合约选择 |
| 合约 | 两个区域或城市之间的商品交易关系 |
| 终局倒计时 | 有人达到现金目标后开启的最后冲刺时间 |

## 22. 试玩时最重要的一句话

城市化份额让你赚钱，怪兽和军队让别人少赚钱，卡牌让你改变局势，而公开线索让你猜出谁在背后操作。最后，钱最多的人赢。
