# Weather v1 Balance Report

> Evidence class: deterministic sample/simulation evidence generated by `WeatherBalanceReportV1Bench`. This is not live production telemetry.

## Scope and method

- The bench records 18 completed samples: three for each of the six canonical Weather v1 definitions.
- Every sample emits forecast, activation, and end lifecycle records. Only completed end records contribute to balance aggregates.
- Durations are centered on the current definition catalog values; deltas and losses are fixed simulation observations, not claims about production players.
- Economic values are aggregate estimates. No player identity or index, owner, exact cash, hand/discard, card identity, hidden monster target or weights, AI plan, save state, or camera state is collected.

## Reading the sample

Negative route-revenue and economic deltas are simulated losses. `Monster target changed` reports only whether a public decision outcome changed; no target identity, candidate weights, or AI reasoning is present. The report is deterministic so changes between runs represent code or fixture changes rather than random sampling noise.
