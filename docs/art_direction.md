# Art Direction

The vertical slice should read as a high-end sci-fi board-game table rather than a debug overlay. The first pass uses procedural UI shapes only:

- Dark table surface with restrained blue/green/gold signal colors.
- Planet region plates in the center, with space visible outside the board boundary.
- Cards use physical-object language: cost badge, route band, lamp, state border, and drag lift.
- Monster pressure uses warning magenta/red only when threat is active.
- Damage uses cracks, red edge, and floating deltas instead of gore or realistic destruction.
- Public track remains anonymous and compact; details move to the inspector.

No Blizzard art, icons, card backs, fonts, text, or trademarks are used.
