# Commercial Playability Gate

本页定义当前阶段的“真人首局可玩性”硬闸门。它不是最终商业化标准，也不替代长期平衡模拟；它专门保护一个没读长规则的人进入真实牌桌后，能看见主要桌面、手牌、牌轨、目标和下一步按钮。

## 当前必须证明的事

- 使用真实 `scenes/main.tscn` 和真实 RuntimeGameScreen，不能只检查菜单、静态 demo 或 showcase。
- 在 `1280×720`、`1600×960`、`1920×1080` 下，TopBar、匿名卡牌/事件时间轴、星球牌桌、右侧详情、底部玩家板、手牌架和行动区都必须可见。
- 牌轨必须是薄轨道，不能像长棒子一样压缩星球或遮住手牌。
- 底部竞价区必须像桌游桌边控件：标题使用“牌桌竞价/报价”这类动作词，不在主桌常驻文案里反复解释匿名机制。
- 星球/地图必须保持主视觉权重，玩家第一眼要知道这是一张“星球赌桌”。
- 星球旁的首局罗盘必须是状态化芯片，不是纯文字：已完成、当前步骤、待办步骤和下一步短句要能一眼区分。
- 运行牌桌必须有基本键盘/手柄分区焦点顺序：顶部状态 → 牌轨 → 星球地图 → 右侧详情 → 手牌 → 当前行动 → 竞价；打开区域牌架时，区域牌架必须插入在右侧详情和手牌之间，形成顶部状态 → 牌轨 → 星球地图 → 右侧详情 → 区域牌架 → 手牌 → 当前行动 → 竞价。
- 星球地图获得焦点后，方向键/手柄方向必须能在区域之间移动选择；每次选区必须让中央星球旋转到目标区域，确认键打开当前区域牌架。
- 牌轨槽位必须能被键盘/手柄聚焦，并可用确认键选择，不能只靠鼠标 hover。
- 手牌卡必须能被键盘/手柄聚焦，并可用确认键完成选择/二次确认出牌。
- 手牌 hover/焦点必须打开左侧桌边大卡预览：使用真实 `CardFace`，文字可换行，不遮挡中央星球主体，不靠 tiny hand text 承担完整读牌。
- 区域牌架里的市场卡必须能被键盘/手柄聚焦，并可用确认键完成预览和尝试购买。
- 战役桌面必须保持单主 CTA：当前目标一句话，一个最重要按钮；复杂规则留在规则页或详情页。
- 当玩家连续求助或卡住时，主桌必须切到强卡住态：目标光框脉冲、Coach 只显示一句最短操作、主 CTA 仍是“定位下一步”，不能伪完成目标。
- 首局 Coach 的主 CTA 必须绑定本地真人席位；玩家查看 AI 公开档案时，Coach 也不能把首召、开牌架、买牌等操作打到 AI 身上。
- 首局 Coach 的主 CTA 成功后，5 秒内必须进入强焦点反馈：目标框脉冲，标签使用“最短｜目标｜动作”，并优先指向刚完成动作的结果区，例如打开牌架后看区域牌架、首召后看行动区。
- 玩家 UI 不得泄漏隐藏信息，包括对手现金、对手手牌、AI 私有计划、真实归属字段、AI 评分拆解。
- 首局动作链至少要能完成：选择推荐区域 → 首召怪兽 → 建第一城 → 打开区域牌架 → 买第一张区域牌 → 打出第一张非起始牌 → 查看顶部牌轨 → 打开经济总览，并同时保持“手牌/牌架/牌轨/GDP”的概念清楚可见。
- 首局引导进度必须语义准确：起始手牌和首召怪兽不能被误判成“已买第一牌”或“已打出第一张普通牌”；真实区域购牌要有独立计数。
- 首局 Coach 自动买牌时必须通过教学补给 gate：候选牌必须可购买、非起始牌、无需目标询问、买入后立刻可打；若怪兽可达牌架没有合格牌，只允许补入低复杂度现金型教学牌，不能随机买到一张暂时完全用不了的牌把玩家卡住。
- 首局第一张普通牌打出前，怪兽赌局这类全场冻结系统必须延后，不能抢走“买第一牌 → 看手牌 → 打第一牌”的基础教学焦点。
- 首局出牌后必须回到经济理解：Coach 能打开经济总览，让玩家看到 GDP、商品和商路如何转成钱。
- 经济总览首屏必须像桌游玩家板，而不是开发报表：先给 KPI，再给三条下一步决策路线（扩GDP、护商路、压竞争），每条只用公开/当前玩家可见信息生成，不得显示 AI 内部计划、对手真实现金、真实手牌或隐藏业主。

## 自动化验收

运行：

```powershell
& "..\tools\godot-4.6.2\Godot_v4.6.2-stable_win64_console.exe" --headless --path . --script res://tests/commercial_playability_gate_test.gd
```

这个测试会实例化真实主场景，进入战役运行路径，并检查布局、CTA、隐藏信息禁词和首局动作链。

- 不证明最终美术、音效和动效已经商业化。
- 不证明 30 分钟完整局已经平衡。
- 不证明 AI 已经达到重型桌游水平。
- 不证明所有手柄型号和平台映射都已经做过真人有头复验。
