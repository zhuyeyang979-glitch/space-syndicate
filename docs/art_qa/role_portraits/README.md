# Role portrait QA

The final contact sheet is produced from the 48 validated PNGs:

- 24 `front.png`
- 24 `side_inward.png`

Each PNG must be 512×768 RGBA with a transparent background. Model selection,
model SHA-256, license SHA-256, hidden nodes, material changes, attachments,
camera, and output paths are recorded in the portrait manifest.
