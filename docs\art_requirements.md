# Art Requirements Extracted From 太空辛迪加美工.xlsx

The workbook contains a single populated sheet with this production list:

- 规则书封面
- 宣传海报
- 名片
- 两只怪兽模型
- 通用技能牌 UI
- 战斗技能牌 UI
- Buff 牌 UI
- 怪兽牌 UI
- 怪兽基础战斗牌 UI
- 伤害 token（1 血和 5 血两种）
- 判断指示物背面美工（代表四个玩家）
- 判断指示物正面美工（破坏、保留、特殊各一）
- 有效移动 token
- 高利贷 token
- 玩家指导牌的流程图
- 筹码美工（1、2、5）
- 战斗结果 token（蓝色怪兽和红色怪兽）
- 行动轨美工

The current Godot prototype uses generated UI placeholders for these categories.
